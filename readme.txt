Sorts the My Sites listings in a WordPress Multisite installation alphabetically.

== Description ==

An intuitive and easy-to-use plugin that sorts the "My Sites" listings in a wordpress multisite installation.

= All features =

* Activate or deactivate sorting with a simple slider.
* No other settings required

== Installation ==

1. Go to Plugins -> Add New section from your network admin account.
2. Search for Xoo Sortio.
3. Download and install plugin Xoo Sortio.
4. Network Activate the plugin.
5. Go to the 'Xoo' menu and 'Xoo Sort' submenu to activate sorting.
5. Your sites are now sorted in alphabetical order.

You can also install this plugin manually:

1. Download the plugin's ZIP archive and unzip it.
2. Copy the unzipped xoo-sort folder to the /wp-content/plugins/ directory.
3. Activate the plugin through the Plugins menu in WordPress.

== Uninstalling ==

Simply deactivate the plugin from the installed plugins page. There is no risk in deactivating/deleting the plugin.
